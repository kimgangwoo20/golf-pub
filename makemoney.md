Golf Pub 앱 → 기존 예약 플랫폼 API → 골프장

직접 제휴 X, 플랫폼을 통해 연동

플랫폼	골프장 수	특징
XGOLF	360여 개	사업제휴 채널 운영
골프존 티스캐너	370여 개	골프존카운티 제휴
카카오골프예약	다수	무료 수수료 정책
사업제휴 문의로 API 연동을 요청하면 됩니다. 다만 초기 앱은 트래픽이 적어 거절당할 수 있습니다.

방법 2: Golf Pub만의 차별화 (가장 현실적)
Golf Pub은 **"골프장 예약 앱"이 아니라 "골프 소셜 앱"**입니다. 기존 앱들과 직접 경쟁하지 말고:

기존 앱: 골프장에서 예약을 받아줌 (B2B)
Golf Pub:  골퍼끼리 모임을 만들고, 함께 갈 골프장을 고름 (소셜)

예약은 외부 링크로 연결하고, Golf Pub은 "사람 모으기"에 집중:

1. 부킹 생성: "2/15 토요일 4명 모집"
2. 사람 모임: 채팅으로 골프장 결정
3. 예약 연결: 카카오골프예약/XGOLF 링크로 이동
4. 라운드 후: 펍에서 뒤풀이 → 리뷰 작성

방법 3: 제휴 골프장 소수로 시작
10~20개 골프장과 직접 제휴 → 성과 보여주고 → 점점 확장

수익 모델 (현실적 순서)
순서	모델	예상 수익	난이도
1	멤버십 구독	월 9,900~19,900원	낮음 (이미 구현됨)
2	중고거래 수수료	거래액의 3~5%	낮음 (이미 구현됨)
3	광고 (골프용품, 골프장)	CPM/CPC	중간
4	예약 수수료	건당 5,000~10,000원	높음 (제휴 필요)
5	기업 단체 예약 대행	건당 수만원	높음
핵심 조언
기존 시장 (카카오, XGOLF)은 이미 예약 수수료 0원 경쟁을 하고 있습니다.
카카오는 예약 수수료를 안 받고 멤버십으로 수익을 냅니다.

Golf Pub이 예약 수수료로 수익을 내기는 매우 어렵습니다.
대신 **소셜 기능 (부킹 모임 + 채팅 + 펍 뒤풀이)**이 차별점이고,
멤버십 + 중고거래 수수료가 현실적인 첫 수익원입니다.


현실적으로 가능한 방법
방법 A: 딥링크로 연결 (지금 바로 가능)
Golf Pub에서 골프장 선택
↓
"예약하기" 버튼 터치
↓
카카오골프예약 앱이 열림 (해당 골프장 페이지로)

Golf Pub의 현실적 수익 로드맵
[지금]
├── 멤버십 구독료 (월 9,900~19,900원)
├── 중고거래 수수료 (3~5%)
└── 골프장 정보 + 외부 예약 앱 연결 (수익 없지만 사용자 확보)

[사용자 1,000명+]
├── 배너 광고 (골프용품, 골프웨어)
├── 펍/레스토랑 제휴 광고
└── 소규모 골프장 제휴 시작

[사용자 10,000명+]
├── 골프장 직접 예약 + 수수료
├── 기업 단체 예약 대행
└── 카카오VX/XGOLF에 제휴 제안 가능

핵심: 지금은 예약 수수료보다 "사용자 모으기"가 우선입니다.
사용자가 모이면 골프장이 먼저 제휴를 요청하게 됩니다.
